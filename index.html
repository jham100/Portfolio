Projects

Crosspoint Evaluations Expert Dashboard

I was contracted by Crosspoint Evaluations to develop a dashboard interface for medical experts to view upcoming and past appointments, patient records and other pertinent information.
I developed this web application with HTML, Javascript (Vanilla, jQuery), CSS (Custom, Bulma), Google Sheets API, Google Apps Script and other G Suite technologies.

Lilypad Rentals

Lilypad rentals is an app designed to match people with homes. The goal of this app was to simplify and expedite the rental search process for prospective renters, and to provide a simple and effective holistic platform for property owners and managers.
My team and I built this app using the MERN (MongoDB, Express, React.js, Node.js) stack.

Womxn Festival

Womxn festival is a hypothetical music / culture festival created by myself and Sophie Kim. I used HTML, CSS, jQuery and Bootstrap in the development of this project.

BitBuzz

Bitbuzz is a web application featuring news and interactive bitcoin price charts. My small team and I used HTML, CSS, Bootstrap, JavaScript, JQuery, Firebase and the Google News API in the development of this site.
